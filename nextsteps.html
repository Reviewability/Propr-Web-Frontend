Github PR Feedback

Instructions on enabling pull request feedback for your repo

Next steps

You have successfully logged in.

Please fill feedback when you close pull requests:

  • When you close pull requests, a link will be posted by our bot as a comment
  • A feedback form will be presented to you upon clicking the link
  • You will be redirected back to the pull request once you submit the feedback form
    • Note: If you don't see the link, refresh your page

Try the alternative setup methods if there was any issue logging in.

  • For private repositories:

    • To add a collaborator go to Settings, then Webhooks from your repository
    • Add prfeedback as a collaborator by using the user search box at the bottom of the page.
    • Continue setting up the webhook as described for public repositories in the next section

  • For public repositories:

    • Add a webhook by going to Settings, then Webhooks from your repository
    • Set the payload URL to the address http://chennai.ewi.tudelft.nl:60002/webhook
    • Set the content type to application/json
    • Ensure that the only data that is being sent over is that related to Pull Requests. To see more about how to setup webhooks, see this page
